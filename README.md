# T-Plus

수능 및 학교별 맞춤 내신 영어 시험지를 효율적으로 제작할 수 있는 IT 솔루션

## Install dependencies

```bash
npm install
```

## Run web & server

```bash
cp .env.default > .env

npm start
```

## Run test

```bash
npm test
```

## Lint and fix all

```bash
npm run lint
```

## Build

```bash
npm run build
```
